### Prerequisites 

To run the slack-bot you need:

- Go installed
- Add appropriate API keys to a .env file within the program as shown below:
```
SLACK_BOT_TOKEN=<YOUR_TOKEN_HERE>
SLACK_APP_TOKEN=<YOUR_TOKEN_HERE>
```
